# Acode JS to HTML Converter Plugin

A simple plugin for [Acode](https://acode.app) that takes your current JavaScript code and wraps it in a functional HTML5 template.

## Features
- **One-click conversion**: Adds a transformation icon to the header.
- **Auto-logging**: Captures `console.log` and displays it directly in the generated HTML.
- **Keyboard Shortcut**: `Ctrl-Alt-H`.

## Installation via URL
1. Go to **Settings > Plugins**.
2. Tap the **+** icon and select **Local**.
3. Select **URL**.
4. Paste the direct link to the `plugin.zip` from the [Releases](link-to-your-github-releases) page.

## Development
To modify this plugin, clone the repo, zip the `main.js` and `plugin.json`, and install the zip file locally in Acode.
